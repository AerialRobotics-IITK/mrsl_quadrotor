l_quadrotor
==========    
package of simulation for mrsl UAV
### Requirements
 - `ros` (indigo)
 - `gazebo` (2.2)
 - `gazebo_ros` (2.2)
 - `quadrotor_msgs`
              
### mrsl_quadrotor_launch
the package for launch demo
```           
$cd ./launch  
$roslaunch gazebo.launch
$roslaunch test.launch
```
